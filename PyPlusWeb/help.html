How to use the 'code menu'

The code menu is an important part in this editor.
You can run your scripts using it, or search for contents.
But you need to install pylint, autopep8, and html-linter by typing: pip install pylint html-linter autopep8
Consider changing pip to pip3 if needed


If you have Poetry, you can try poetry install

How to CUSTOMIZE the editor

This editor uses a JSON file as settings file.
Here's a quick description

- File type (file_type)


all - all files
py - python files
xml - xml files
txt - plain text and reST

- Font (font)


Since this text editor is written in Tk,
the font need to be a tk font.
Format: fontname size

- Theme (theme)


This setting is for the app's theme
And again, since it is written in Tk,
It MUST be a ttk theme.
Click here for a list of all themes.
Note that some themes only support specific platforms
